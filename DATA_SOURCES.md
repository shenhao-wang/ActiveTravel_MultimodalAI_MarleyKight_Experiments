# Data Sources

## 1. Tampa Municipal Code (Chapter 27 — Zoning and Land Development)

- **Location:** `raw_data/all_zoncode/`, `raw_data/Ch26_27.*`
- **Source:** City of Tampa Municode library
- **URL:** https://library.municode.com/fl/tampa/codes/code_of_ordinances?nodeId=COOR_CH27ZOLADE
- **Format:** Plain text (scraped), Word doc, Markdown
- **Access:** Public, no account required

## 2. Tampa Bay Regional Travel Survey (TBRTS-HTS, 2019)

- **Location:** `raw_data/HTS_Data_files/`
- **Source:** Tampa Bay Regional Transportation Stakeholders (TBRTS)
- **URL:** Not publicly available
- **Format:** CSV (trip, person, household, day, vehicle tables), Excel codebook, PDF reports
- **Access:** Not publicly available
- **Files:** 7 data tables + codebook + 2 PDF reports

## 3. American Community Survey 5-Year Estimates (2015--2019)

- **Location:** Generated by `code/01_zoning_scraping/censusdata.ipynb` -> `processed_data/acs_blockgroup_tampa.csv`
- **Source:** U.S. Census Bureau ACS via `census` Python package
- **API key required:** Yes (Census API key)
- **Geography:** Block groups in Hillsborough County, FL (FIPS 12-057)

## 4. TIGER/Line Shapefiles (2025)

- **Location:** `raw_data/tl_2025_12_bg/`, `raw_data/tl_2025_us_county/`
- **Source:** U.S. Census Bureau TIGER/Line
- **URL:** https://www.census.gov/geographies/mapping-files/time-series/geo/tiger-line-file.html
- **Format:** Shapefile (.shp/.dbf/.prj/.shx)
- **Files:**
  - `tl_2025_12_bg` — Florida block group boundaries
  - `tl_2025_us_county` — U.S. county boundaries

## 5. City of Tampa GIS — Zoning Districts

- **Location:** `raw_data/shapefiles/Zoning/`
- **Source:** City of Tampa Open Data / GIS
- **URL:** https://city-tampa.opendata.arcgis.com/datasets/tampa::zoning-district/explore?location=28.057943%2C-82.485339%2C11
- **Format:** Shapefile
- **Files:** `Zoning_District.*`

## 6. City of Tampa GIS — Existing Land Use

- **Location:** `raw_data/shapefiles/ExistingLandUse/`
- **Source:** City of Tampa Open Data / GIS
- **URL:** https://planhillsborough.org/wp-content/uploads/2026/02/ELU_Generalized_POLY_20260601.zip
- **Format:** Shapefile
- **Files:** `ExistingLandUse_20251201.*`

## 7. City of Tampa GIS — Future Land Use & City Boundary

- **Location:** `raw_data/shapefiles/city_tampa_shapefiles/`, `raw_data/shapefiles/Tampa_City/`
- **Source:** City of Tampa Open Data / GIS
- **URL:** https://planhillsborough.org/wp-content/uploads/2023/04/TPA_FLU_Shapefiles.zip
- **Format:** Shapefile, .lyr
- **Files:** `TA_FLU.*`, `Tampa_FLU.lyr`, `tampa_bg.*`

## 8. Hillsborough County Land Development Code

- **Location:** `raw_data/HILLSBOROUGH_COUNTY___LAND_DEVELOPMENT_CODE.txt`
- **Source:** Hillsborough County Municode library
- **URL:** https://library.municode.com/fl/tampa/codes/code_of_ordinances?nodeId=10132
- **Format:** Plain text (scraped)
- **Note:** County-level code, used for context/comparison — not the primary Tampa city code
